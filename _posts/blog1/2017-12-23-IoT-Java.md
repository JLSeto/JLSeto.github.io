## Background
This is a generic IoT demo using Java to connect to the sensor.

## TLDR / How it Works:
Arduino waits for a connection and a signal. Sends that data over Bluetooth to Java Server. Server processes data and displays over GUI.

## Hardware Used:
ArduinoUno
Bluetooth Module
Laptop

## Software Used:
Java
