## Background
This project is an IoT demo with Arduino101 and Node.js.  

## TLDR / How it Works:
CC2560 Sensortag sends data over bluetooth to an App. App sends data over wifi to the server. Server processes data and displays on webpage.

## Hardware Used:
Arduino101
Laptop

## Software Used:
Node.js, Noble
HTML, CSS, JavaScript
